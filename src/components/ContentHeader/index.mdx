# ContentHeader

- 标题组件。

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| showLine | 下划线 | boolean | false |
| color | 标题前图标颜色 | string | #398EEB |
| padding | 内边距 | string | '20px 30px 10px 20px' |
| stripVisible | 控制蓝条显隐 | boolean | true |
| fontWeight | 标题字体 | string | '600' |
